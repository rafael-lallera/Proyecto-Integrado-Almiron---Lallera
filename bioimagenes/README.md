## Librería de Procesamiento de Imágenes Médicas

Este proyecto consiste en el desarrollo de una librería en Python orientada al procesamiento, análisis y visualización de imágenes médicas, aplicando principios de Programación Orientada a Objetos (POO). La solución permite trabajar con distintos tipos de imágenes, como tomografías, radiografías y termografías, incorporando funcionalidades como filtrado, mejora de calidad, análisis de intensidades y visualización específica según el dominio.

La arquitectura del sistema está diseñada de forma modular y extensible, incluyendo gestión de metadatos y un historial de transformaciones que garantiza la trazabilidad de los procesos aplicados sobre cada imagen.

## Instalación del entorno

Para reproducir el entorno de desarrollo:

conda env create -f environment.yml
conda activate miEntornoPDA
